# 02. 응급 처치 교육 웹사이트(First Aid LMS)

## 프로젝트 개요
응급 상황 대처 능력을 향상시키기 위한 온라인 응급 처치 교육 웹사이트입니다.  
강의 수강, 시험, Unity 기반의 응급 처치 게임 등을 통해 학습자가 재미있게 응급 처치를 익히도록 하는 것을 목표로 했습니다.

본 프로젝트는 DW 아카데미에서 진행한 2차 팀 프로젝트이며,  
저는 관리자 페이지 전반의 프론트엔드 개발을 담당했습니다.

---

## 기술 스택
- HTML
- CSS
- JavaScript
- Chart.js

---

## 주요 구현 범위

### ▶ 관리자 페이지 전체 UI 개발
- 전체 관리자 레이아웃(header / sidebar / footer / 공통 UI) 제작
- CSS 모듈화 및 파일 구조 정리
- 열리고 닫히는 사이드바 토글 기능 구현

### ▶ 강의 관리
- 강의 목록 / 등록 / 수정 UI 개발
- 파일 업로드 UI(영상 업로드 영역, 자막 입력 영역)
- 강의별 상세정보 폼 구조화

### ▶ 대시보드 개발
- 신규 가입자 수 라인 차트 구현
- 강의 수료율 막대 그래프 구현
- 핵심 지표 네 가지 요약 카드 구현
