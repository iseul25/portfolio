# 02. 응급 처치 교육 웹사이트(First Aid LMS)

## 프로젝트 개요
응급 상황에서 필요한 기초 응급처치 지식을 효율적으로 학습할 수 있는 온라인 교육 웹사이트입니다.
강의 수강과 퀴즈를 통해 핵심 지식을 체계적으로 학습하고,
Unity 기반의 시뮬레이션형 게임을 통해 실제 상황 대응 절차를 보조적으로 체험할 수 있도록 구성했습니다.

본 프로젝트는 DW 아카데미에서 진행한 2차 팀 프로젝트이며,  
저는 관리자 페이지 전반의 프론트엔드 개발을 담당했습니다.

---

## 기술 스택
- HTML
- CSS
- JavaScript
- Chart.js

---

## 주요 구현 범위

### ▶ 관리자 페이지 전체 UI 개발
- 전체 관리자 레이아웃(header / sidebar / footer / 공통 UI) 제작
- CSS 모듈화 및 파일 구조 정리
- 열리고 닫히는 사이드바 토글 기능 구현

### ▶ 강의 관리
- 강의 목록 / 등록 / 수정 UI 개발
- 파일 업로드 UI(영상 업로드 영역, 자막 입력 영역)
- 강의별 상세정보 폼 구조화

### ▶ 대시보드 개발
- 신규 가입자 수 라인 차트 구현
- 강의 수료율 막대 그래프 구현
- 핵심 지표 네 가지 요약 카드 구현
