# 01. 중등 역사 교육 웹사이트 (History Education LMS)

## 프로젝트 개요
중등 학생들을 위한 역사 교육 학습 콘텐츠를 제공하는 웹사이트입니다.  
강의 시청, 퀴즈, 포인트 상점 등 핵심 기능을 통해 학습 효과를 높이는 데 목적이 있습니다.

본 프로젝트는 DW 아카데미에서 진행한 1차 팀 프로젝트이며,  
저는 **사용자 + 관리자 화면 전체 프론트엔드 개발**을 담당했습니다.

---

## 기술 스택
- **HTML**
- **CSS**
- **JavaScript**

---

## 주요 기능

### ▶ 사용자 페이지
- 강의 목록 및 시청 페이지
- 퀴즈 풀이 기능
- 출석 체크 기능
- 포인트 획득 → 포인트 상점 사용

### ▶ 관리자 페이지
- 회원 등록 / 수정 / 삭제
- 강의 등록 / 수정 / 삭제
- 퀴즈 콘텐츠 등록 / 수정 / 삭제
- 전체 콘텐츠 관리 UI 제작
