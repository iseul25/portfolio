# 01. 역사 한 걸음(History Education LMS)

## 프로젝트 개요
중학교 학생들을 위한 온라인 역사 교육 콘텐츠를 제공하는 웹사이트입니다.  
강의 시청, 퀴즈 풀이, 포인트 상점 등을 통해 학습 흥미를 높이는 것을 목표를 했습니다.

본 프로젝트는 DW 아카데미에서 진행한 1차 팀 프로젝트이며,  
저는 사용자 / 관리자 화면 전반의 UI 구현을 담당했습니다.

---

## 기술 스택
- HTML
- CSS
- JavaScript

---

## 주요 기능

### ▶ 사용자 페이지
- 강의 목록 UI 구성
- 강의 상세 페이지 기본 구조 제작
- 퀴즈 페이지의 화면 레이아웃 및 UI 흐름 구현
- 포인트 / 출석 체크 화면 UI 제작

### ▶ 관리자 페이지
- 회원 / 강의 / 퀴즈 관리 UI 구성
- 콘텐츠 등록 / 수정 / 삭제 화면 UI 프로토타입 제작

---

## 프로젝트 코드 공개 여부
본 프로젝트는 웹 개발을 처음 배우는 단계에서 진행한 초기 작품으로,  
현재 기준으로 코드 품질이 미흡하며 소스 코드는 비공개 처리했습니다.
